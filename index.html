01. 웹사이트 목적

“오늘 뭐 먹지?” 고민을 빠르게 해결하는 음식 추천 사이트

목적

수업이 끝난 뒤 식사 메뉴를 결정하는 데 드는 시간을 줄이고, 학생이 자신의 상황에 맞는 음식을 쉽게 찾을 수 있도록 돕는 것을 목표로 합니다.

목표

공개해서 제출할 수 있는 형태의 웹사이트를 만들고, 대학생활에서 자주 겪는 "뭐 먹지" 상황을 주제로 정리했습니다.

02. 주요 기능

기획 의도에 맞게 정리한 핵심 콘텐츠

A

상황별 추천

빠르게 먹기, 든든하게 먹기, 야식처럼 실제 학생 생활에 맞춘 선택 기준을 제시합니다.

B

카테고리 구성

한식, 중식, 양식, 분식으로 구분해 원하는 취향의 메뉴를 쉽게 탐색할 수 있습니다.

C

시각적 정보

대표 이미지, 간단한 설명, 특징을 함께 보여 주어 한눈에 비교하기 쉬운 구조로 구성했습니다.

D

반응형 UI

모바일과 PC 환경에서 모두 보기 편하도록 그리드와 카드 레이아웃을 적용했습니다.

03. 제작 과정

기획부터 구현, 출판까지 한눈에 보이는 흐름

1

기획 정리

대상 사용자를 대학생으로 설정하고, 메뉴 결정의 불편함을 해결하는 방향으로 목적을 확정했습니다.

2

화면 구성

인트로, 소개, 기능, 추천 메뉴, 제작 정보가 자연스럽게 이어지는 정보 구조를 설계했습니다.

3

HTML/CSS 구현

섹션별 의미 구조를 살리고 카드형 레이아웃, 그라데이션, 버튼, 배지 등을 활용해 완성도를 높였습니다.

4

Github Pages 출판

정적 파일만으로도 배포할 수 있게 정리했고, Github Pages에 올리는 과정까지 포함했습니다.
